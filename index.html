CSS Selectors and Specificity

Selector Utility Specificity
element {} Standard selector, selects one element of choice and modifies it within the curly braces. Most common form of specificity, revolves around most recent changes
element, element {} Selects all elements declared separated by a comma (,) and modifies them within the curly braces. Most common form of specificity, revolves around most recent changes
element element {} Selects the nested element within the first element and modifies it within the curly braces. More specific than standard selectors due to nesting
.class {} Selects all elements that hold a certain class, and modifies them within the curly braces. More specific than standard selectors and nesting due to class attribute selection
#id {} Selects all elements that hold a certain id, and modifies them within the curly braces. More specific than all other selectors!
Thank you for visiting! Hope you learned more about css selectors from here. In addition to standard selector nesting, you can also nest elements with classes: (element class {}) or (class element) or select all specificied elements that have a certain class with chaining (element.class)
